# 数字简谱音乐生成器

南方科技大学《数字信号处理》课程项目

## 项目简介

基于MATLAB的数字简谱音乐生成系统，实现了从数字简谱到音频文件的完整转换流程。项目结合了数字信号处理理论与音乐合成技术，支持多种乐器音色模拟和参数调节。

## 主要功能

1. **简谱解析**：支持完整简谱语法（音符、升降调、八度、时值、和弦）
2. **频率计算**：基于十二平均律精确计算音符频率
3. **波形生成**：生成时域波形，支持包络衰减控制
4. **音色模拟**：模拟钢琴、小提琴、长笛等9种乐器音色
5. **图形界面**：提供直观的GUI界面，实时调节参数
6. **音乐播放**：生成音乐可实时播放或保存为音频文件
7. **随机生成**：根据不同风格自动生成简谱片段

## 技术特点

- 基于十二平均律的频率计算
- 加性合成实现音色模拟
- 包络衰减控制音符时间特性
- 支持和弦与多音符合成
- 实时波形与频谱显示

## 使用方法

1. 在MATLAB中运行 `run_app.m` 启动应用程序，或直接启动
2. 在左侧输入简谱（支持单音、和弦、休止符等）
3. 调整调号、乐器、速度、拍号等参数
4. 点击"生成音乐"按钮创建音频波形
5. 播放生成音乐或保存为WAV/MP3文件

## 简谱语法示例

- 单音：`5. 6. 7. 1' 7 6 5 4 3 4 5-`
- 和弦：`(135) (246)`
- 休止符：`0 0-`
- 八度标记：`'`（高八度），`,`（低八度）
- 升降调：`#`（升号），`b`（降号）
